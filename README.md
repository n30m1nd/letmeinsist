# Let Me Insist
A simple bash script to automate information gathering about linux systems

*Credits to [blog.g0tmi1k.com](https://blog.g0tmi1k.com/2011/08/basic-linux-privilege-escalation/)*

## Help
* Just run: `chmod +x lmi.sh; ./lmi.sh`
